# Accès au bot ClosLamartine Monitor

Guide rapide pour obtenir l'accès au bot de monitoring.

---

## 1. Trouver le bot

1. Ouvre **Telegram**
2. Dans la recherche, tape **@closlamartineBot** ou ouvre directement : [t.me/closlamartineBot](https://t.me/closlamartineBot)
3. Clique sur **Démarrer** ou envoie `/start`

---

## 2. Obtenir ton Chat ID

Envoie l'une de ces commandes au bot :

| Commande | Effet |
|----------|-------|
| `/myid`  | Le bot te renvoie ton Chat ID |
| `/start` | Idem que `/myid` |
| `/help`  | Liste des commandes disponibles |

Ces commandes fonctionnent **sans accès préalable**.

---

## 3. Demander l'accès

1. Copie ton **Chat ID** (le nombre que le bot t'a envoyé)
2. Envoie-le à Denis (ou à la personne qui gère le monitoring)
3. Une fois ton ID ajouté à la config, tu recevras une confirmation

---

## 4. Après activation

Une fois ton accès activé, tu pourras utiliser toutes les commandes, par exemple :

- `/status` — Statut complet (prod, staging, serveur)
- `/status_prod` — Production uniquement
- `/status_staging` — Staging uniquement
- `/help` — Rappel des commandes
